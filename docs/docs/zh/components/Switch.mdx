## 基础用法

用于需要表示开关状态或者两种状态直接的切换。

```js live=true
() => <Switch label="Label" />
```

## 文字

带有文字的开关选择器

```js live=true
() => <Switch okText="开启" offText="关闭" />
```

## 禁用状态

```js live=true
() => <Switch disabled />
```

## 按钮模式

```js live=true
() => <Switch variant="button" label="Edit Yaml" />
```
