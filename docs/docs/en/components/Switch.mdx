## Basic Usage

Switch is used for switching between two opposing states.

```js live=true
() => <Switch label="Label" />
```

## Text description


```js live=true
() => <Switch okText="Turn on" offText="Turn off" />
```

## Disabled status

```js live=true
() => <Switch disabled />
```

## variant

```js live=true
() => <Switch variant="button" label="Edit Yaml" />
```
