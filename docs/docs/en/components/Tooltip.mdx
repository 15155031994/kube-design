## Usage

<Button>按钮</Button>

```js live=true
<Group>
  <Button color="secondary" radius="xl">KubeSphere</Button>
  <Tooltip content="KubeSphere">
    <TestWrapper>按钮</TestWrapper>
  </Tooltip>

  <Tooltip content="KubeSphere">
    <Text>按钮</Text>
  </Tooltip>
</Group>
```
